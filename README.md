# ShiftSolver
A small solver for shift rostering for many-to-one scenario. Employees specify their indivdual availablity and their desired working hour range in the excel sheet. The solver tries to fulfill all constraints and creates a solved excel file containing only the assigned shifts of the best solution.

# Usage
Just drag and drop a filled-in template. The solver start automatically end terminates at a set score or when a time limit is reached
